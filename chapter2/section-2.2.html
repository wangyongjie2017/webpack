2.2 基本选择器

1 元素选择器

元素选择器基于元素名选取元素。

一个基本的例子，读取页面的所有<p>元素：

$('p')

使用元素选择器的实际例子：

用户点击按钮后，隐藏所有p元素：

<button>点我</button>
<p>需要隐藏的元素</p>

$(document).ready(function(){
    // 绑定button元素的点击事件
    $("button").click(function(){
        // button按钮点击后，因此所有`p`元素
        $("p").hide();    
    });
});

2. id选择器

jQuery 可以通过id选择器，查找出定义了id属性的HTML元素。

基本语法：$('#test')

使用id选择器的实际例子：

HTML：

<button>点我</button>
<div id="test">需要隐藏的元素</div>

jQuery：

$(document).ready(function(){
    $("button").click(function(){
        $("#test").hide();
    });
});

3. class选择器

jQuery 类选择器可以通过指定的 class 查找元素。

基本语法： $('.test')

使用类选择器的实际例子：

HTML：

<button>点我</button>
<div class="test">需要隐藏的元素</div>

jQuery：

$(document).ready(function(){
    $("button").click(function(){
        $(".test").hide();
    });
});
