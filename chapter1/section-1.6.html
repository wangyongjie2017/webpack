1.7 jQuery对象与DOM对象

讲义

DOM对象，表示的是文档对象模型（Document Object Model），一般情况可以表示为一棵树。

jQuery对象，是通过jQuery包装DOM对象后所生成的对象。

所以，jQuery返回的对象不等于DOM对象

如何规范定义变量的风格：

通过jQuery对象变量名，其名称前面加上$符号，而DOM对象则不需要。

例如：

// jQuery对象
var $name = $('#name');

// DOM对象
var name = document.getElementById('name');

jQuery对象转换为DOM对象

因此，jQuery对象不能直接调用DOM对象的方法和属性，例如innerHTML属性。如果需要从jQuery转换为DOM对象，可通过以下方法转换：

// jQuery对象
var $name = $('#name');

// 返回为DOM对象
var name = $name[0];

// 调用DOM对象的innerHTML属性
console.log(name.innerHTML);

DOM对象转换为jQuery对象

同样的，DOM对象也可以通过一定方法，转换为jQuery对象，其方法非常简单。

// DOM对象
var name = document.getElementById('name');

// 把DOM对象转换为jQuery对象
var $name = $(name);

// 输出HTML代码
console.log($name.html());
