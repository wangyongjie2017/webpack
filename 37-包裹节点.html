jQuery课程

包裹节点

jQuery提供了将某个节点用其他标记包裹起来的有用方法。

  • wrapAll()。该方法会将所有匹配的元素用一个元素来包裹。
  • wrapInner()。该方法将每一个匹配的元素的子内容（包括文本节点）用其他结构化的标记包裹起来。
  • wrap()。把所有匹配的元素用其他元素的结构化标记包裹起来。
  • unwrap()。这个方法将移出元素的父元素。这能快速取消 .wrap()方法的效果。匹配的元素（以及他们的同辈元素）会在DOM结构上替换他们的父元素。

wrap

这种包装对于在文档中插入额外的结构化标记最有用，而且它不会破坏原始文档的语义品质。这个函数的原理是检查提供的第一个元素（它是由所提供的HTML标记代码动态生成的），并在它的代码结构中找到最上层的祖先元素－－这个祖先元素就是包裹元素。当HTML标记代码中的元素包含文本时无法使用这个函数。因此，如果要添加文本应该在包裹完成之后再行添加。

案例：

把所有的段落用一个新创建的div包裹起来

$("p").wrap("<div class='wrap'></div>");

用原先div的内容作为新div的class，并将每一个元素包裹起来

HTML:

<div class="container">
    <div class="inner">Hello</div>
    <div class="inner">Goodbye</div>
</div>

jQuery：

$('.inner').wrap(function(){
    return '<div class="' + $(this).text() + '" />';
});

结果：

<div class="container">
    <div class="Hello">
        <div class="inner">Hello</div>
    </div>
    <div class="Goodbye">
        <div class="inner">Goodbye</div>
    </div> 
</div>

wrapAll()

用一个生成的div将所有段落包裹起来

$("p").wrapAll("<div></div>");

用一个生成的div将所有段落包裹起来

$("p").wrapAll(document.createElement("div"));

wrapInner()

包裹子内容

$('strong').wrapInner("<b></b>")

把所有段落内的每个子内容加粗

$("p").wrapInner(document.createElement("b"));

用原先div的内容作为新div的class，并将每一个元素包裹起来

HTML:

<div class="container">
    <div class="inner">Hello</div>
    <div class="inner">Goodbye</div>
</div>

jQuery:

$('.inner').wrapInner(function() {
    return '<div class="' + $(this).text() + '" />';
});

结果：

<div class="container">
    <div class="inner">
        <div class="Hello">Hello</div>
    </div>
    <div class="inner">
        <div class="Goodbye">Goodbye</div>
    </div>
</div>
